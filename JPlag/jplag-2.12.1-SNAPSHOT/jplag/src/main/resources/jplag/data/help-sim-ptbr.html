Ranking de Equivalências

JPlag compara programas tentando cobrir um dos programas com trechos (preferencialmente grandes) de outros programs. A cobertura de um programa é definida como a razão do código fonte coberto com seu tamanho total.

Ranking de equivalências pela similaridade média

A similaridade média é definida como a média das coberturas de ambos os programas.
Essa é a similaridade padrão que funciona na maioria dos casos: Equivalências com uma similaridade alta indica que os programas funcionam de uma maneira muito semelhante.

Ranking de equivalências pela similaridade máxima

A similaridade máxima é definida como a máxima similaridade da cobertura de ambos os programas.
Esse ranking é especialmente útil quando programas são muito distintos em tamanho. Isso pode acontecer quando partes inúteis de código são inseridas com o intuito de enganar um programa que busca plágios.
